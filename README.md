# РОСЧАТ LITE

## Введение

Мессенджер «РОСЧАТ LITE» это удобное корпоративное решение, которое открывает Вам функционал групповых бесед, информационных каналов, безопасных звонков и видеоконференций для Вашей команды. Установка своего сервера позволяет Вам получить доступ локальной версии мессенджера. Хранение личных данных пользователей, обработка и рассылка сообщений, организация вызов и видеоконференций будет реализована на установленном Вами сервере, что является важным фактором безопасности для корпоративного общения. Вы так же можете ознакомится с функциями полной версии приложения перейдя по [ссылке](https://www.minicom.ru/resheniya/korporativnyy-messendzher-roschat/).

## Установка

Для начала установки Вам необходимо выбрать целевую машину на которой будет установлено серверное приложение. На данный момент серверное решение поддерживает следующие типы ОС:
* Ubuntu 24 
* Debian 13 
* Fedora 43
* RedOS 8 
* AlmaLinux 9

После подготовки чистового образа с выбранной ОС, для начала установки Вам необходимо выполнить установочный скрипт. Вы можете скачать его и выполнить установку вручную или использовать команды curl или wget для установки.

```
curl -o- https://raw.githubusercontent.com/roschat/roschat-server/refs/heads/main/install_podman.sh | bash
```
```
wget -qO- https://raw.githubusercontent.com/roschat/roschat-server/refs/heads/main/install_podman.sh | bash
```

> [!NOTE]
> Вовремя установки следующие сетевые порты, будут открыты и добавлены как исключение в службу сетевой безопасности:
> * 80 — HTTP
> * 443 — HTTPS
> * 3478, 3479 — TURN 

## Настройка

После выполнения установки у Вас есть возможность подключиться к серверу используя в качестве пути сетевой адрес с указанием порта 1110. При первом подключении, так как SSL сертификат еще не был установлен, адрес необходимо добавить в исключение браузера. Пример записи адреса панели администратора:
```
https\\:192.168.1.1:1110
```
Логин и пароль по умолчанию:
```
admin
123456
```
> [!IMPORTANT]
> После успешного подключения необходимо сменить пароль [администратора безопасности](https://github.com/roschat/roschat-server/wiki/%D0%9F%D0%B0%D0%BD%D0%B5%D0%BB%D1%8C-%D0%B0%D0%B4%D0%BC%D0%B8%D0%BD%D0%B8%D1%81%D1%82%D1%80%D0%B0%D1%82%D0%BE%D1%80%D0%B0).

Затем необходимо добавить администратора системы для доступа к системе управления. Вы можете ознакомится с возможностями настройки сервера в приведенной [документации](https://github.com/roschat/roschat-server/wiki).

> [!CAUTION]
> Внимание! Для безопасной работы сервера вам необходимо выполнить настройку [сертификата](https://github.com/roschat/roschat-server/wiki/%D0%9D%D0%B0%D1%81%D1%82%D1%80%D0%BE%D0%B9%D0%BA%D0%B8#%D1%81%D0%B5%D1%80%D1%82%D0%B8%D1%84%D0%B8%D0%BA%D0%B0%D1%82) в системе управления, а так же выполнить > > регистрацию доменного имени в сети интернет. Без этой настройки соединение с сервером является небезопасным для использования сервера в публичной сети интернет.

## Подключение

Для подключения к серверу web-клиентом используете адрес сервера в адресной строке браузера. Так же доступна мобильная версия приложения и версия компьютерного клиента:
### iOS
* [AppStore](https://apps.apple.com/ru/app/%D1%80%D0%BE%D1%81%D1%87%D0%B0%D1%82/id1161341380)
### Android
* [Google Play](https://play.google.com/store/apps/details?id=com.itsoft.lan.irbis)
* [AppGallery](https://appgallery.huawei.com/#/app/C114033507)
* [RuStore](https://apps.rustore.ru/app/com.itsoft.lan.irbis)
### Компьютерный клиент
* [Windows x32](https://firelink.me/download/client-windows-x86-latest)
* [Windows x64](https://firelink.me/download/client-windows-x64-latest)
* [Linux (rpm)](https://firelink.me/download/client-rpm-based-x64-latest)
* [Linux (dep)](https://firelink.me/download/client-deb-based-x64-latest)
* [Linux (appImage)](https://firelink.me/download/client-appimage-x64-latest)
* [MacOS](https://firelink.me/download/client-macos-x64-latest)

При первом подключении пользователь имеет возможность самостоятельно зарегистрироваться на сервере. Администратор системы может выполнить ограничения для этого функционала в разделе [дополнительно](https://github.com/roschat/roschat-server/wiki/%D0%9D%D0%B0%D1%81%D1%82%D1%80%D0%BE%D0%B9%D0%BA%D0%B8#%D1%80%D0%B5%D0%B3%D0%B8%D1%81%D1%82%D1%80%D0%B0%D1%86%D0%B8%D1%8F-%D0%BD%D0%BE%D0%B2%D1%8B%D1%85-%D0%BF%D0%BE%D0%BB%D1%8C%D0%B7%D0%BE%D0%B2%D0%B0%D1%82%D0%B5%D0%BB%D0%B5%D0%B9).
